# Project description

1. A simple implementation of the classic 2048 puzzle game, inspired by the original 2048 game by Gabriele Cirulli. Use arrow keys to move tiles: up, down, left, right. When two tiles with the same number collide, they merge into one. Each move spawns a new tile (usually a 2 or 4). The game ends when no moves are left.

2. Live preview available at:
    - [DEMO LINK](https://hardaira.github.io/Game_2048/)

3. Technologies used:

HTML5: Provides the structure of the game, including the grid layout, tiles, and overall page structure with semantic and accessible markup.

CSS3: Handles the visual design of the game, including tile styles, animations, layout, and responsive behavior across different screen sizes.

JavaScript: Powers the game logic and interactivity, including tile movement, merging mechanics, score tracking, and dynamic updates using modern JavaScript features.

4. To set up the project locally:
    - Clone the repository:
    git clone https://github.com/hardaira/Game_2048.git
    cd Game_2048
    -Install dependencies:
    npm install
    -Run the project locally:
    npm start
